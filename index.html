JAWABAN

  1. Pertanyaan seputar Tag
    1. Tag pada HTML adalah elemen-elemen yang digunakan untuk menandai dan mengatur konten di dalam halaman web. Tag dikelilingi oleh tanda kurung sudut < > dan biasanya terdiri dari nama elemen HTML di antara kurung tersebut. Seperti : <p>, <div>, <a>, dan <body>, Setiap tag memberi tahu browser bagaimana menampilkan atau menginterpretasikan teks, gambar, atau elemen lainnya di dalam halaman.

    2. Tag pembuka adalah tag yang menandai awal dari suatu elemen. Formatnya adalah: <tagname>. Contohnya: <p> untuk menandai paragraf. Tag penutup adalah tag yang menandai akhir dari elemen tersebut. Formatnya adalah: <tagname>. Misalnya: <p> untuk menutup paragraf yang dimulai oleh <p>.

      Tidak semua jenis tag di HTML memiliki penutup. Ada beberapa tag self-closing yang tidak memerlukan penutup karena mereka tidak mengandung konten. Tag ini biasanya digunakan untuk elemen yang tidak memiliki isi, seperti <img> (gambar) atau <br> (baris baru). Contoh format self-closing tag adalah <img /> atau <br />, meskipun dalam HTML5, hanya <img> atau <br> sudah dianggap cukup.

  2. Pertanyaan seputar Element
    1. Dalam HTML, elemen adalah blok penyusun dari sebuah halaman web yang mencakup tag pembuka, konten (jika ada), dan tag penutup. Elemen inilah yang digunakan untuk membentuk struktur halaman web, seperti paragraf, link, gambar, tabel, dan sebagainya. Contoh elemen lengkap adalah: ( <p>This is a paragraph.</p> ) Elemen tersebut terdiri dari tag pembuka <p>, konten "This is a paragraph.", dan tag penutup </p>.

      • Tagname

        Tagname adalah nama dari elemen HTML yang digunakan untuk mengidentifikasi jenis elemen tersebut. Misalnya, dalam elemen <p<, tagname-nya adalah p, yang menandakan paragraf.

      • Attribute

        Attribute adalah informasi tambahan yang ditambahkan ke tag untuk memberikan properti atau perilaku tertentu kepada elemen tersebut. Attribute ditulis di dalam tag pembuka dan terdiri dari pasangan nama dan nilai. Misalnya, dalam elemen <a href="https://example.com" >, href adalah attribute yang menentukan tujuan link. Contoh lain dari elemen dengan attribute : <img src="image.jpg" alt="Description of image" > Di sini, src adalah attribute yang menunjuk pada lokasi gambar, dan alt adalah attribute yang memberikan teks alternatif untuk gambar.

      • Content

        Content adalah teks, gambar, atau elemen lain yang berada di antara tag pembuka dan tag penutup. Misalnya, dalam elemen <p>This is content</p>, teks "This is content" adalah konten dari elemen tersebut.


    3. Ya, elemen dapat ditumpuk atau bersarang satu sama lain. Elemen yang bersarang berarti ada satu elemen di dalam elemen lainnya. Jika elemen ditumpuk dengan benar, browser akan menampilkannya sesuai dengan struktur. Namun, jika penumpukan tidak tepat (misalnya tag tidak tertutup), akan terjadi kesalahan rendering atau tampilan yang tidak sesuai.
  3. Anchor
    1. Anchor adalah elemen HTML yang digunakan untuk membuat hyperlink. Elemen ini memungkinkan pengguna untuk mengklik teks atau elemen lainnya untuk menuju ke halaman lain, bagian lain dari halaman yang sama, atau sumber daya eksternal. Tag anchor ditulis dengan tag <a>.

    2. Pendefenisian

      Sebuah anchor ditulis dengan tag <a> dan attribute href, yang menentukan tujuan link. Contoh : <a href="https://example.com">Klik di sini Dalam contoh ini, teks "Klik di sini" akan menjadi hyperlink yang, jika diklik, akan membawa pengguna ke "https://example.com".

    3. Pengaplikasian

      Navigasi ke halaman lain Dengan href yang berisi URL eksternal. Navigasi ke bagian lain dari halaman yang sama Menggunakan anchor dengan tanda # diikuti ID elemen yang dituju, seperti: <a href="#section">Menuju Bagian> </a> <div id="section">Ini adalah bagian yang dituju</div>

  4. Pertanyaan seputar Multimedia
    1. Text
      • - Tag untuk memanipulasi konten teks :
        • <p> : membuat paragraf. < i > : Membuat teks miring. < u > : Menggarisbawahi teks. < strong > : Memberi penekanan tebal pada teks, biasanya untuk SEO. < em > : Memberi penekanan miring pada teks, biasanya untuk penekanan logis.
        • <i> : Membuat teks miring.
        • <u> : Menggarisbawahi teks.
        • <strong> : Memberi penekanan tebal pada teks, biasanya untuk SEO.
        • <em> : Memberi penekanan miring pada teks, biasanya untuk penekanan logis.

      • - Tag untuk memanipulasi struktur konten teks:
        • <p> : Membuat paragraf. <br>: Menyisipkan baris baru. <h1> sampai <h6>: Heading level 1 hingga 6 untuk judul atau subjudul. <blockquote>: Menampilkan teks sebagai kutipan blok. <pre>: Menampilkan teks dalam format asli (termasuk spasi dan baris). <hr>: Menyisipkan garis horizontal sebagai pembatas.
    2. Image
      • Penyebab Gambar Tidak Ditampilkan :
        • - Kesalahan pada URL : URL yang salah atau gambar tidak tersedia di lokasi yang ditentukan oleh src. - Format gambar tidak didukung : Browser tidak mendukung format gambar (misalnya, file .webp di browser lama). - Kesalahan penulisan tak : Kesalahan sintaksis, seperti penulisan atribut src atau penempatan tanda kutip.

      • Teks yang menggantikan gambar :
        • Jika gambar tidak tampil, teks dari atribut alt akan muncul sebagai pengganti. Contoh : <img src="image.jpg" alt="Gambar tidak dapat ditampilkan" >
    3. Audio

      - Untuk memasukkan audio dari sumber web, gunakan tag <audio> dengan atribut src yang mengarah ke file audio online. Contoh : < audio controls > <source src="https://example.com/audio.mp3" type="audio/mpeg" > < /audio >

    4. Video

      - Atribut autoplay mungkin tidak berjalan karena browser mencegah pemutaran otomatis untuk meningkatkan pengalaman pengguna (terutama jika suara menyala). - Solusi : Agar autoplay berfungsi, tambahkan atribut muted sehingga video dimulai tanpa suara, karena banyak browser hanya mengizinkan autoplay jika video dimulai dalam mode tanpa suara. Contoh : <video autoplay muted > <source src="video.mp4" type="video/mp4" > < /video >
